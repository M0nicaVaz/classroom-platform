### Setup

**Clone the project and access the folder**

```bash
git clone https://github.com/M0nicaVaz/classroom-platform.git && cd classroom-platform
```

**Follow the steps below**

```bash
# Install the dependencies
$ npm or yarn install

# Run the web server
$ npm run dev
```
